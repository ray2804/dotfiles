Write

Synopsis

Generic writer for all types of IOObjects.


Description

Generic writer for all types of IOObjects. Writes one of the input objects into a given file.


Input

  • object: expects: IOObject

Output

  • object:

Parameters

  • object file: Filename of the object file.
  • output type: Indicates the type of the output
  • continue on error: Defines behavior on errors
