Map Clustering on Labels

Synopsis

This operator converts the cluster attribute into an prediction attribut, choosing the best fitting pairs between cluster and label.


Description

This operator estimates a mapping between a given clustering and a prediction. It adjusts the given clusters with the given labels and so estimates the best fitting pairs.
