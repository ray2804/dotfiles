Cluster Density Performance

Synopsis

Delivers a performance based on cluster densities.


Description

This operator is used to evaluate a non-hierarchical cluster model based on the average within cluster similarity/distance. It is computed by averaging all similarities / distances between each pair of examples of a cluster.
