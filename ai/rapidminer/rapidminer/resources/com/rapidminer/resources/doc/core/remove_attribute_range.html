Remove Attribute Range

Synopsis

This operator removes a range of features.


Description

This operator removes the attributes of a given range. The first and last attribute of the range will be removed, too. Counting starts with 1.


Input

  • example set input: expects: ExampleSetMetaData: #examples: = 0; #attributes: 0

Output

  • example set output:
  • original:

Parameters

  • first attribute: The first attribute of the attribute range which should be removed
  • last attribute: The last attribute of the attribute range which should be removed
