Clear Log

Synopsis

Clears a table generated by a ProcessLog operator.


Description

This operator can be used to clear a data table generated by a ProcessLogOperator.


Input

  • through 1:

Output

  • through 1:

Parameters

  • log name: The name of the log table which should be cleared.
  • delete table: Indicates if the complete table should be deleted. If this is not checked, only the entries will be deleted.
