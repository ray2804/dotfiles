Superset

Synopsis

This operator gets two example sets and adds new features to each of both example sets so that both example sets consist of the same set of features.


Description

This operator gets two example sets and adds new features to each of both example sets so that both example sets consist of the same set of features. This set is the union or the superset of both original feature sets. The values of the new features are set to missing. This operator only works on the regular attributes and will not change, add, or otherwise modify the existing special attributes.


Input

  • example set 1: expects: ExampleSet
  • example set 2: expects: ExampleSet

Output

  • superset 1:
  • superset 2:
