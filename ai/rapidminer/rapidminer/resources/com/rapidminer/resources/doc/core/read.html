Read

Synopsis

Generic reader for all types of IOObjects.


Description

Generic reader for all types of IOObjects. Reads an IOObject from a file.


Input


Output

  • output:

Parameters

  • object file: Filename of the object file.
  • ignore type: Indicates if the execution should be aborted if type of read object does not match selected type.
  • io object: The class of the object(s) which should be saved.
