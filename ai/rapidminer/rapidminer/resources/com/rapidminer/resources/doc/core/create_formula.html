Create Formula

Synopsis

Generates a formula from a given model for models which are capable of producing formulas.


Description

This operator extracts a prediction calculation formula from the given model and stores the formula in a formula result object which can then be written to a file, e.g. with the ResultWriter operator. Please note that not all RapidMiner models provide a formula.


Input

  • model: expects: Model

Output

  • formula:
  • model:
