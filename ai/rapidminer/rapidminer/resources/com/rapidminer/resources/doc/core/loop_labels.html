Loop Labels

Synopsis

Performs its inner operators for each label found in input example set.


Description

Performs the inner operator for all label attributes, i.e. special attributes whose role name starts with "label". In each iteration one of the multiple labels is used as label. The results of the inner operators are collected and returned. The example set will be consumed during the iteration.
